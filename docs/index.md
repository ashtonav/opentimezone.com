# Overview

<img src="/img/favicon.svg" alt="Description" width="105"></img>

OpenTimezone is an open source, free, public API that converts time between different time zones.

## Features

- **Open Source**<br/>
  The source code is available on [GitHub](https://github.com/ashtonav/opentimezone).

- **Free, Public API**<br/>
  Access a fully open, cost-free API at [api.opentimezone.com](https://api.opentimezone.com) without needing API keys or facing usage limits.

- **Self-Hosting Option**<br/>
  Host OpenTimezone on your own infrastructure for total control over uptime, security, and scalability. Please see the [Self-Hosting](self-hosting) section.
